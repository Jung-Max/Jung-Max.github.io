мқҙлІҲ нҸ¬мҠӨнҠём—җм„ңлҠ” мҠӨн”„л§Ғ л¶ҖнҠёмҷҖ MariaDBлҘј м—°кІ°н•ҳлҠ” кІғм—җ лҢҖн•ҙм„ң мӮҙнҺҙліёлӢӨ.

MariaDB м„Өм№ҳ

https://downloads.mariadb.org/лЎң л“Өм–ҙк°Җ MariaDBлҘј лӢӨмҡҙл°ӣлҠ”лӢӨ.

м„Өм№ҳ лІ„м „мқ„ мӮ¬мҡ©н•ңлӢӨл©ҙ .msi / л¬ҙм„Өм№ҳ лІ„м „мқҖ .zipмқ„ лӢӨмҡҙл°ӣлҠ”лӢӨ.

м„Өм№ҳмӢң NextлҘј лҲ„лҘҙкі  root кі„м • мһ…л ҘмӢң passworldлҘј м„Өм •н•ңлӢӨ.

мңҲлҸ„мҡ°мҰҲ м„ңл№„мҠӨлЎң лҸҷмһ‘н•ҳл©° м„ңл№„мҠӨ мқҙлҰ„мқҖ кё°ліёк°’мқё вҖңMySQLвҖқ мқ„ к·ёлҢҖлЎң мӮ¬мҡ©н•ңлӢӨ.

вҖңEnable netwokringвҖқ мқ„ мІҙнҒ¬н•ңлӢӨ. нҸ¬нҠёлҠ” кё°ліёк°’мқё вҖң3306вҖқмқ„ к·ёлҢҖлЎң мӮ¬мҡ©н•ңлӢӨ.

к·ё нӣ„ м„Өм№ҳлҘј мҷ„лЈҢн•ҳкі  мңҲлҸ„мҡ°мҰҲ мӢңмһ‘лІ„нҠјм—җм„ң мһҳ м„Өм№ҳк°Җ лҗҳм—ҲлҠ”м§Җ нҷ•мқён•ңлӢӨ.

SLQ к°ңл°ңнҲҙ DBeaver

MariaDBлҠ” мһҗмІҙ нҲҙмқё HeidSQLмқҙ мһҲм§Җл§Ң м—¬кё°м„ңлҠ” DBeaverм—җ лҢҖн•ҙм„ң м„ӨлӘ…н•ңлӢӨ.

лӢӨмҡҙлЎңл“ңлҠ” https://dbeaver.io/download/м—җм„ң Communuty Editionмқ„ лӢӨмҡҙл°ӣлҠ”лӢӨ.

DBeaver лҘј мӢӨн–үн•ҳл©ҙ мўҢмёЎм—җ Database Navigator м°Ҫмқҙ мһҲкі , м „мӣҗм—°кІ° м•„мқҙмҪҳм—җ л§Ҳмҡ°мҠӨлҘј мҳ¬л Ө лҶ“мңјл©ҙ <мғҲ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°> л¬ёкө¬лҘј нҷ•мқён•  мҲҳ мһҲлӢӨ.

м•һм„ң лӢӨмҡҙл°ӣмқҖ MariaDBлЎң лҚ°мқҙн„°лІ мқҙмҠӨлҘј м„ нғқн•ҳл©ҙ, Connection Settings м°Ҫмқҙ лң¬лӢӨ.

MariaDB м„Өм№ҳн•ҳл©° м„Өм •н•ң л№„л°ҖлІҲнҳёмҷҖ лҸҷмқјн•ҳкІҢ мӮ¬мҡ©мһҗмқҙлҰ„, л№„л°ҖлІҲнҳёлҘј мһ…л Ҙн•ҳкі  м ҖмһҘн•ңлӢӨ.

  • DBeaver л©”лүҙ мӨ‘ нҢҢмқј > мғҲлЎң мһ‘м„ұ мқ„ нҒҙлҰӯн•ҳл©ҙ лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ° л§ҲлІ•мӮ¬ м°Ҫмқҙ лң¬лӢӨ. DataBase Navigator м—җм„ң л§Ҳмҡ°мҠӨ мҡ°нҒҙлҰӯ > Create > Connection лҘј н•ҙлҸ„ лҗңлӢӨ. лӢЁм¶•нӮӨлҠ” Ctrl+N.

  • лҚ°мқҙн„°лІ мқҙмҠӨ MariaDB м„ нғқ

  • DB м •ліҙмһ…л Ҙ л°Ҹ port мһ…л Ҙ нӣ„ мғқм„ұ мҷ„лЈҢ

SpringBootмҷҖ MariaDB м—°кІ°

мҠӨн”„л§Ғл¶ҖнҠёмҷҖ л§ҲлҰ¬м•„DBлҘј м—°кІ°н•ҳкё° мң„н•ң м„Өм •мқҙлӢӨ.

л§ҲлҰ¬м•„ DBлҘј м„Өм •н•ҳкё° мң„н•ң gradleмқ„ м„Өм •н•ңлӢӨ.

м—¬кё°м„ңлҠ” implementation group: 'org.mariadb.jdbc', name: 'mariadb-java-client', version: '2.4.1'мқ„ dependenciesм—җ л„Јм–ҙмӨҢмңјлЎңм„ң л§ҲлҰ¬м•„DBмқҳ м„Өм •мқ„ н•ңлӢӨ.

м„Өм • нӣ„ gradleмқ„ мҡ°нҒҙлҰӯ нӣ„ refeshлҘј мһҠм§Җ м•ҠлҠ”лӢӨ.

src/main/resourcesм—җ мһҲлҠ” application.propertiesлҘј м„Өм •н•ңлӢӨ.

м—¬кё°м„ң localhost:3306/t_boardм—җм„ң t_board лҢҖмӢ  м„Өм •лҗң dbмқҳ мқҙлҰ„мқ„ м Ғм–ҙмӨҖлӢӨ.

DatabaseConfiguration нҒҙлһҳмҠӨ мғқм„ұ

л©”мқё нҢЁнӮӨм§Җ л°‘м—җ configuration нҢЁнӮӨм§ҖлҘј мғқм„ұн•ҳкі  DatabaseConfiguration нҒҙлһҳмҠӨлҘј л§Ңл“ лӢӨ.

м•„лһҳ мҪ”л“ңмҷҖ к°ҷмқҙ мһ‘м„ұн•ңлӢӨ.

в‘  application.propertiesлҘј мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ м„Өм • нҢҢмқјмқҳ мң„м№ҳлҘј м •н•ҙ мӨҖлӢӨ.

в‘Ў application.propertiesм—җ м„Өм •н–ҲлҚҳ лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖл Ё м •ліҙлҘј мӮ¬мҡ©н•ҳлҸ„лЎқ м§Җм •н•ңлӢӨ.

@ConfigurationProperties м–ҙл…ён…Ңмқҙм…ҳм—җ prefixк°Җ spring.datasource.hikariлЎң м„Өм •лҗҳм—Ҳкё° л•Ңл¬ём—җ spring.datasource.hikariлЎң мӢңмһ‘н•ҳлҠ” м„Өм •мқ„ мқҙмҡ©н•ҙм„ң нһҲм№ҙлҰ¬CPмқҳ м„Өм •нҢҢмқјмқ„ л§Ңл“ лӢӨ.

в‘ў м•һм—җм„ң л§Ңл“  нһҲм№ҙлҰ¬CPмқҳ м„Өм •нҢҢмқјмқ„ мқҙмҡ©н•ҙм„ң лҚ°мқҙн„°лІ мқҙмҠӨмҷҖ м—°кІ°н•ҳлҠ” лҚ°мқҙн„° мҶҢмҠӨлҘј мғқм„ұн•ңлӢӨ. м—¬кё°м„ңлҠ” лҚ°мқҙн„° мҶҢмҠӨк°Җ м •мғҒм ҒмңјлЎң мғқм„ұлҗҳм—ҲлҠ”м§Җ нҷ•мқён•ҳкё° мң„н•ҙм„ң лҚ°мқҙн„° мҶҢмҠӨлҘј м¶ңл Ҙн–ҲлӢӨ.
